= Agents Module

The agents module covers the `Agent` and `AgenticWorkforce` custom resources — deploying agents from templates or custom images, configuring their tools and sub-agents, and grouping them into discoverable workforces.

== How-to guides

* xref:agents:how-to-guides/create-template-agent.adoc[] — deploy an agent using an operator-managed template image.
* xref:agents:how-to-guides/create-custom-image-agent.adoc[] — deploy an agent using your own container image.
* xref:agents:how-to-guides/propagate-headers.adoc[] — forward HTTP headers from incoming A2A requests to MCP tool servers.
* xref:agents:how-to-guides/delegate-to-subagent.adoc[] — configure sub-agents and pick `tool_call` vs. `transfer` interaction.
* xref:agents:how-to-guides/create-workforce.adoc[] — group entry-point agents into an `AgenticWorkforce`.

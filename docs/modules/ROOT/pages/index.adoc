= Agent Runtime Operator

The Agent Runtime Operator is a Kubernetes operator that provides a unified, framework-agnostic way to deploy and manage agentic workloads on Kubernetes. It uses Custom Resource Definitions (CRDs) to abstract away framework-specific configurations, simplifying the deployment and management of agents across different frameworks.

== What is the Agent Runtime Operator?

The Agent Runtime Operator enables you to:

* Deploy and manage agentic workloads on Kubernetes
* Abstract framework-specific configurations using CRDs
* Wire agents with observability stacks and model routers
* Support multiple agent frameworks (Google ADK, Flokk, Autogen)

== Key Features

* **Framework Agnostic**: Support for multiple agent frameworks
* **Kubernetes Native**: Built using Kubernetes operator patterns
* **Custom Resources**: Simplified agent deployment through CRDs
* **Observability**: Built-in integration with monitoring stacks
* **Scalable**: Deploy multiple agent replicas
